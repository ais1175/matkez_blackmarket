# Black Market
#### Black market script with leveling system for QBOX and ESX frameworks
<img width="1920" height="1080" alt="blackmarket" src="https://github.com/user-attachments/assets/e0ed1869-0083-43da-9026-6f5d08cb016b" />


### Features
- Ordering system with pickup locations
- Leveling system
- Randomized item prices
- Randomized pickup locations
- Configurable currency per black market

### Requirements
- QBOX or ESX framework
- [ox_lib](https://github.com/CommunityOx/ox_lib)
- [ox_inventory](https://github.com/CommunityOx/ox_inventory)
- [ox_target](https://github.com/CommunityOx/ox_target) or [sleepless_interact](https://github.com/Sleepless-Development/sleepless_interact)
